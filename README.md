## 🚀 Déploiement sur GitHub Pages

### Étapes de Publication

1. **Créer un Repository GitHub**
   ```bash
   # Initialiser Git dans le dossier du projet
   git init
   git add .
   git commit -m "Initial commit: Sirius landing page"
   ```

2. **Pousser vers GitHub**
   ```bash
   # Ajouter l'origine remote (remplacer par votre URL)
   git remote add origin https://github.com/VOTRE-USERNAME/sirius-website.git
   git branch -M main
   git push -u origin main
   ```

3. **Activer GitHub Pages**
   - Aller dans Settings > Pages
   - Source: Deploy from a branch
   - Branch: main / (root)
   - Cliquer "Save"

4. **Accéder au Site**
   - URL: `https://VOTRE-USERNAME.github.io/sirius-website`

### Configuration Recommandée

- **Repository Name**: `sirius-website` ou `VOTRE-USERNAME.github.io` (pour un site principal)
- **Visibilité**: Public (requis pour GitHub Pages gratuit)
- **Branch**: `main`
